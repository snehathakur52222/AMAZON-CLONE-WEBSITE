🛒 Amazon Clone

A responsive Amazon homepage clone built using HTML, CSS, JavaScript, and jQuery. This project replicates the look and feel of Amazon's landing page while adding interactive features such as product search, cart management, smooth scrolling, and dynamic UI effects.

🚀 Features

Frontend UI

- Amazon-inspired navigation bar
- Search bar with category selection
- Language selector
- Account & Orders section
- Shopping cart icon
- Product cards and promotional sections
- Responsive footer layout

Interactive Features

- Product search using JavaScript and JSON data
- Add products to cart
- Dynamic cart counter
- Back-to-top button with smooth scrolling
- Navbar shadow effect on scroll
- Language selection alerts
- Account option alerts
- Deals section interaction

Data Handling

- Product data stored in JSON format
- Dynamic filtering and searching
- Cart updates without page refresh

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- jQuery
- Font Awesome
